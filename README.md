# TTG Customer Feedback App

## Chạy trên máy
```bash
npm install
npm run dev
```

## Đưa lên Vercel
1. Tạo tài khoản và đăng nhập Vercel.
2. Upload thư mục này lên GitHub.
3. Trong Vercel, chọn **Add New Project**.
4. Import repository GitHub chứa app.
5. Vercel sẽ tự nhận Vite:
   - Build Command: `npm run build`
   - Output Directory: `dist`
6. Bấm **Deploy**.
7. Sau khi deploy xong, bạn sẽ có link dạng `https://ten-du-an.vercel.app`
